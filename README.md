#How to use
#Recommend line thickness 100, and adjust vertical ratio to choose location. Side margins are there just for case
#Choose a font to use, and click store for the plugin to remember it.
#There is an autoshrink in case the contents don't fit within padding. It will shrink items proportionately within ratio
#You can choose top image and adjust size by dragging the white squares but the plugin replies faster when you use the image ratios
#Middle symbol is for the white line symbol, use image ratio for fast respond, also there is a center function that will center your symbol
#You can save templates. Make a template, name it, save it. Next time you need it, find it in the dropdown, and load it.
#I guess that is it?
#font recs: Garamond italics for title, Futura for author and series
